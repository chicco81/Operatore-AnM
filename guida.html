🔐 Accesso e installazione
1
Apri chicco81.github.io/Operatore-AnM su Safari (iPhone) o Chrome (Android)
2
Inserisci Matricola e Password → tocca Accedi
3
Primo accesso? Tocca Password dimenticata? → inserisci matricola + data di nascita
📲 Installala sulla Home (consigliato):
iPhone: Safari → tasto Condividi (□↑) → "Aggiungi alla schermata Home" → icona ANM blu
Android: Chrome → ⋮ → "Aggiungi a schermata Home"
📅 Calendario turni
Navigazione mesi
Scorri sinistra/destra per cambiare mese. Domeniche in fucsia, festività in rosa.
Assegnare un turno
Tocca un giorno → pannello dettaglio → ✏️ Assegna turno → scegli dalla lista.
Range multi-giorno
Tocca il primo giorno, poi il secondo → range arancione → scegli il turno → assegnato a tutti i giorni.
Rimuovere un turno
Tocca il giorno → 🗑 Rimuovi → conferma.
🕐 Fasce orarie multiple NUOVO

Quando hai più turni in un giorno (cambio linea, servizio spezzato), aggiungi fasce extra. Appaiono direttamente sotto il nome del turno nel calendario.

1
Tocca il giorno nel calendario
2
Nel pannello scorri fino a 🕐 Fasce orarie
3
Tocca ＋ Aggiungi fascia oraria
4
Inserisci l'orario (es. 06:40-09:35) e la linea (es. S5)
5
Tocca ✕ per rimuovere una fascia
Come appare nel calendario:
🌅
S5/1
06:40-09:35
↳ 10:30 180/1
↳ 14:00 633/3
← Il turno principale (S5/1)
con le fasce extra (↳) sotto,
visibili subito nel calendario
💡 Le fasce appaiono anche nel pannello dettaglio del giorno e nell'esportazione mensile.
📥 Importa quindicinale PDF

Carica il PDF del quindicinale ANM e i turni vengono importati automaticamente nel calendario.

1
Vai nel tab 📥 Importa
2
Tocca Carica PDF quindicinale e seleziona il file
3
Controlla i turni riconosciuti e tocca Importa nel calendario
24/Mar → 633/3 montata 05:20 smonto 11:20
25/Mar → S2/1 montata 05:05 smonto 11:15
26/Mar → 165/3 montata 05:10 smonto 11:30
27/Mar → spezzato apertura
✅ Parser verificato sul quindicinale ANM reale. Legge esattamente: linea/turno macchina, orario di montata (Pres.) e orario di smonto calcolato preciso (Pres. + Durata).
🔄 Bacheca Cambi
Pubblicare un cambio
Tab 🔄 Cambi → 📢 Pubblica → seleziona data → il turno viene preso automaticamente → aggiungi WhatsApp → Pubblica. Appare in tempo reale su tutti i telefoni.
Contattare un collega
Tab 🔄 → 📋 Bacheca → tocca 💬 WhatsApp → messaggio precompilato.
⚠️ Richiede internet. Solo matricole ANM (14000-25000) o INT (50000-55000).
📞 Rubrica ANM
862 contatti — 4 depositi
🔵 Cavalleggeri Aosta · 🟣 Via Puglie · 🟢 Carlo III · 🟠 San Giovanni
Ricerca
Per cognome, nome, matricola o numero di telefono. Filtro per deposito in alto.
Condividi il tuo turno NUOVO
Tocca un contatto → seleziona la data → tocca 💬 WA → si apre WhatsApp con il tuo turno già scritto (linea, orario, fasce extra).
📤 Esporta e PDF
Condividi turni mensili
Tab 📤 Esporta → condividi via WhatsApp/Telegram con tutte le fasce orarie incluse.
PDF professionale
Tab 📤 → 📄 Esporta PDF mensile → genera un PDF con header ANM blu/giallo, tutti i giorni con turno, orario montata/smonto, linea e fasce extra.
Backup dati
⚙️ Account → Esporta backup → file JSON con tutti i turni. Ripristinalo su un nuovo telefono.
🚍 Flotta ANM in tempo reale
🗺️ Mappa live
Tab 🚍 Flotta → Mappa live: posizione di tutti i bus ANM tramite Infoclick. Se non si carica tocca ↗️ Apri.
🔗 Accesso rapido
Link diretti: ANM GO (mappa GPS), Infoclick (orari fermata), Stato del servizio (interruzioni).
⚙️ Impostazioni

Scuro/chiaro con un tap.
🔔 Notifiche
